Welcome to the Task Management Application

Here you are able to add new task, view task details and more!!

To add new task and view details:
    Click on "Task Creation and Details" and input data in to required fields(First Name, Last Name, Username, Task and Date)
    Data will be inputted into "Task Details" table.

Navigate to the Dashboard to see a visually appealing layout of all the services we provide.

To Register a new user to the system:
    Click on "Register" in the navbar and you will be taken to a page to register.
    Input your First Name, Last Name, Username and Password and click submit.

To Authenticate user information:
    Click on "Authenticate" in the navbar to be taken to the authentication page.
    Enter Username and Password you with to authenticate and click submit.

Thank you for using the Task Management Application.
